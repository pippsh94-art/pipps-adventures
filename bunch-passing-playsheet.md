## Core Concepts

### 1. Stick Concept (vs. Zone Coverage)

**Formation:** 3x1 Bunch  
**Concept:** Horizontal stretch of linebackers

```
H: 12-yard Dig ────────→
Z: 8-yard Stick     ●
X: 6-yard Hitch ●       
Y: 12-yard Comeback ●   

QB Read: Middle linebacker
- If LB drops → throw stick
- If LB sits → throw dig over top
```

**Coaching Points:**
- Z receiver finds soft spot in zone
- H receiver crosses face of safety
- X receiver sits vs. pressure
- Y is hot route vs. blitz

---

### 2. Smash Concept (vs. Man Coverage)

**Formation:** 3x1 Bunch  
**Concept:** High-low corner with pick potential

```
H: Corner route      ●╱╲
Z: Speed out (6 yds) ●→ 
X: Corner route        ╱╲●
Y: 15-yard Out ●→       

QB Read: Corner/Safety
- Single high → throw corner behind safety
- Two high → throw speed out underneath
```

**Coaching Points:**
- Natural pick/rub action in bunch
- Z must get width on speed out
- H and X coordinate releases to avoid collisions
- Timing: 5-step drop

---

### 3. Mills Concept (Flood)

**Formation:** 3x1 Bunch  
**Concept:** Overload zone coverage with levels

```
H: 8-yard Drag →→→→→→●
Z: 12-yard Square-in →●  
X: 18-yard Comeback ●    
Y: 20-yard Go         ●

QB Read: Inside-out progression
1. Check drag first
2. Square-in in window
3. Comeback vs. leverage
```

**Coaching Points:**
- H sets pick for others, then drags
- Z finds hole between LB and safety
- X drives vertical first, breaks back
- Work quickly to avoid coverage rotation

---

### 4. Four Verticals

**Formation:** 3x1 Bunch  
**Concept:** Stretch defense vertically, convert routes

```
H: Seam (read safety) ●
Z: Seam (read safety) ●  
X: Seam (read safety) ●    
Y: Seam               ●

Coverage Conversions:
- Cover 2: All bend to seams at 18-20 yds
- Cover 3: Outside convert to comebacks 12-15 yds  
- Cover 4: Inside seams, outside corners
```

## RPO Package

### Zone + Slant
```
Read: Inside linebacker
- If LB steps to run → throw slant
- If LB sits on slant → hand off zone
```

### Zone + Bubble
```
Count the box (6+ = throw bubble)
H: Bubble screen ●→→→
Z: Block          ●
X: Block          ●
```

### Zone + Stick
```
Read: Middle linebacker leverage
- If LB over top → hand off
- If LB underneath → throw stick behind
```

## Protection Schemes

### 3x1 Protection Rules

| Position | Assignment |
|----------|------------|
| **LT** | Big-on-big with LG, work to LB |
| **LG** | A gap to Mike LB |
| **C** | Identify Mike, work A gaps |
| **RG** | A gap to Mike LB |
| **RT** | Big-on-big with RG, work to LB |
| **RB** | Scan inside-out, pick up blitzer |

### Hot Routes vs. Pressure
- **6 man pressure:** RB picks up 6th rusher
- **Overload blitz:** Convert to quick slants
- **Safety blitz:** Fade route over vacated area

## Game Planning Checklist

### Down & Distance
- **1st & 10:** RPO package, establish rhythm
- **2nd & 7+:** Four verticals, flood concepts  
- **3rd & 3-6:** Stick/smash, pick plays
- **3rd & 7+:** Verticals, deep shots

### Field Position
- **Own 20:** Quick game, safe throws
- **Midfield:** Full concept package
- **Red Zone:** Fades, picks, timing routes
- **Goal Line:** Fade/slant package

### Coverage Identification
- **Cover 2:** Seam routes, intermediate levels
- **Cover 3:** Deep comebacks, underneath
- **Cover 4/Quarters:** Four verticals, flood
- **Man Coverage:** Pick plays, smash concepts
