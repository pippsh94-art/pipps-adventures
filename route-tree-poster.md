## RPO QUICK REFERENCE

### **ZONE + SLANT RPO**
```
Read: Inside Linebacker

    X ● (Route 0: Slant)
      ↙
     QB ● ─── Read ILB
      │       
     RB ● (Zone path)

If ILB:
- Steps up → Throw slant  
- Sits → Hand off zone
```

### **ZONE + BUBBLE RPO**
```
Pre-Snap Count: 6+ in box

    H ●→→→ (Bubble)
    Z ●     (Block)
    X ●     (Block)
        ↓
       QB ● Count box
        │
       RB ● (Zone)

Decision:
- 6+ defenders → Bubble
- 5 or less → Zone run
```

## SITUATIONAL FOOTBALL

### **3RD DOWN CALLS**
| **Distance** | **Primary** | **Secondary** |
|-------------|-------------|---------------|
| **1-3 yards** | Power O | IZ/QB Sneak |
| **4-7 yards** | Stick | Smash |  
| **8+ yards** | Four Verts | Deep concepts |

### **RED ZONE CALLS**
| **Situation** | **Best Plays** |
|--------------|---------------|
| **1st & Goal** | Power O, Fade |
| **2nd & Goal** | Stick, Pick |
| **3rd & Goal** | Slants, Sneak |

### **TWO-MINUTE CALLS**
- Quick slants (stop clock)
- Speed outs (sideline)  
- Comebacks (stop clock)
- Spikes when needed
