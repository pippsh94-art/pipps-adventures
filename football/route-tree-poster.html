Core Concepts — Quick Reference

Stick Concept (vs Zone)

    H ────────→ (Route 2: Dig - 12 yards)
    Z     ●     (Stick: 8-yard settle)
    X ●         (Route 4: Hitch - 6 yards)
        ↓
       QB

Read: Middle Linebacker
- Drops → Throw stick
- Sits → Throw dig over top

Smash Concept (vs Man)

    H    ╱● (Route 7: Corner)

    X   ╱●  (Route 7: Corner)

      QB

Read: Corner/Safety
- 1 High → Corner behind
- 2 High → Speed out underneath

RPO Quick Reference

Zone + Slant RPO

Read: Inside Linebacker

    X ● (Route 0: Slant)
      ↙
     QB ● ─── Read ILB
      │
     RB ● (Zone path)

If ILB:
- Steps up → Throw slant
- Sits → Hand off zone

Zone + Bubble RPO

Pre-Snap Count: 6+ in box

    H ●→→→ (Bubble)
    Z ●     (Block)
    X ●     (Block)
        ↓
       QB ● Count box
        │
       RB ● (Zone)

Decision:
- 6+ defenders → Bubble
- 5 or less → Zone run
