# Zone Blocking Schemes - Play Sheet

## Inside Zone (IZ)

### Formation & Alignment
```
  DE    DT    NG    DT    DE
   |     |     |     |     |
   |     |     |     |     |
  LT----LG----C ----RG----RT
               |
              QB
               |
              RB
```

### Blocking Rules

| Position | Primary | Secondary | Key Points |
|----------|---------|-----------|------------|
| **LT** | Big-on-big w/ LG | Climb to LB if no help needed | Vertical set, don't get beat inside |
| **LG** | Big-on-big w/ C/LT | Work to Mike LB | Get movement, secure B gap |
| **C** | Big-on-big (NT) | Help where needed | Identify Mike pre-snap |
| **RG** | Big-on-big w/ C/RT | Work to Mike LB | Secure B gap, climb together |
| **RT** | Big-on-big w/ RG | Climb to LB if no help needed | Don't get beat inside |

### Running Back Rules
1. **Press the hole** (B gap)
2. **Read the backside** (cut if flow goes away)
3. **Stay low** through contact
4. **Follow double teams** to second level

## Outside Zone (OZ)

### Formation & Alignment
```
  DE    DT    NG    DT    DE
   |     |     |     |     |
   |     |     |     |     |
  LT----LG----C ----RG----RT
               |
              QB → (pitch relation)
               |
              RB → → → (stretch)
```

### Blocking Rules

| Position | Primary | Secondary | Key Points |
|----------|---------|-----------|------------|
| **LT** | Reach block DE | Cut off if no reach | Get helmet outside, create running lane |
| **LG** | Reach block DT | Climb if reached | Lateral step first, then vertical |
| **C** | Back block NG | Help guards | Prevent penetration |
| **RG** | Reach block DT | Climb if reached | Work with center |
| **RT** | Reach block DE | Kick out if necessary | Most critical block |

### Running Back Rules
1. **Get width first** (stretch the defense)
2. **Press the edge** until defense shows
3. **Cut back** when pursuit overruns
4. **Be patient** - let blocks develop

## Gap Schemes (Counter/Power)

### Counter Trey
```
  DE    DT    NG    DT    DE
   |     |     |     |     |
  ←←     |     |    ←←     |
  LT    LG----C     RG    RT
          ↗         ↗
               QB
         ←←←
              RB
```

### Counter Trey Rules

| Position | Assignment | Key Points |
|----------|------------|------------|
| **LT** | Big-on-big, work to backer | Secure inside gap |
| **LG** | Pull, kick out DE | Get depth, look inside-out |
| **C** | Back block | Prevent penetration |
| **RG** | Pull, wrap through hole | Lead blocker for RB |
| **RT** | Big-on-big w/ TE | Double team if needed |

## Power O
```
     TE   DE    DT    NG    DT
      |    |     |     |     |
      ↓    |     |     |     |
  FB--↓-LT----LG----C ----RG
           ↗         ↗    ←←
                QB
              RB
```

### Power O Rules

| Position | Assignment | Key Points |
|----------|------------|------------|
| **LT** | Vertical set vs. DE | Don't get beat inside |
| **LG** | Big-on-big w/ C | Double team nose |
| **C** | Big-on-big w/ LG/RG | Control A gaps |
| **RG** | Pull, kick out LB/DE | Attack inside shoulder |
| **RT** | Cut off backside | Prevent pursuit |
| **FB** | Lead block through hole | Attack inside shoulder |

## Run Concepts Summary

### When to Call Each Play

| Down & Distance | Primary | Secondary | Reasoning |
|----------------|---------|-----------|-----------|
| **1st & 10** | Inside Zone | Power O | Establish rhythm |
| **2nd & 5-7** | Outside Zone | Counter | Create explosive play |
| **3rd & 2-3** | Power O | Inside Zone | Get tough yards |
| **Goal Line** | Power O | QB Sneak | Punch it in |
| **Long Yardage** | Draw | Screen | Keep defense honest |

## Practice Drills

### 1. **Big-on-Big Drill**
- **Setup:** Two linemen vs. two defenders
- **Goal:** Work to linebackers together
- **Key:** Communication and timing

### 2. **Reach Block Drill**
- **Setup:** Lineman vs. defender in B gap
- **Goal:** Get helmet outside shoulder
- **Key:** Lateral step first, then climb

### 3. **Running Back Vision Drill**
- **Setup:** RB with bags/cones as defenders
- **Goal:** Read and cut appropriately
- **Key:** Press hole, then react

### 4. **Full Zone Drill**
- **Setup:** O-line vs. front 7
- **Goal:** Execute scheme rules
- **Key:** Work as unit, climb together

## Game Situation Calls

### Short Yardage (3rd/4th & 2)
1. **Power O** - Double teams + pulling guard
2. **Inside Zone** - Simple, reliable
3. **QB Sneak** - If 1 yard or less

### Red Zone (Inside 20)
1. **Power O** - Create movement
2. **Inside Zone** - Between tackles
3. **Outside Zone** - If defense loads box

### Long Yardage (2nd/3rd & 8+)
1. **Draw** - Let pass rush get up field
2. **Screen** - Use pass rush against them
3. **Check to Pass** - Don't waste down
