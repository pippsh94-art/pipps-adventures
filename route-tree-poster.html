GAME DAY SIGNALS

FORMATION CALLS

  • Point Right → Bunch Right
  • Point Left → Bunch Left
  • Both Hands Up → Trips
  • Fist → Tight Bunch

CONCEPT CALLS

  • “Stick” → Stick Concept
  • “Smash” → Smash Concept
  • “Mills” → Mills/Flood
  • “Four” → Four Verticals

  • “Alert! Alert!” → Hot routes
  • “Check! Check!” → RPO read
